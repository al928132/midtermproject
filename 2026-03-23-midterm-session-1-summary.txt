SESSION 1 SUMMARY — DIG 4503 Midterm (Travel Dream React App)
==============================================================

PRIMARY WORK
------------
Built "Travel Dream" — a React front-end app. Steps 1–5 were implemented
sequentially across session 1. Step 5 was still in progress when context ran out.

TECH STACK
----------
- React 19 + Vite, base path `/midtermproject/` in vite.config.js
- react-router-dom (BrowserRouter, Routes, Route, Navigate, NavLink)
- localStorage for all persistence (auth, wishlist, trips)
- SHA-256 password hashing via crypto.subtle
- Regex form validation (email, password rules)
- React Context (AuthContext) for global auth state
- Custom hooks: useWishlist, useTrips
- CSS custom properties design system (no external UI library)
- Native HTML5 Drag-and-Drop API for trip destination reordering
- Unsplash CDN + Pexels CDN image URLs
- useMemo for filtered destination search
- Modal pattern with Escape key listener and backdrop click-to-close
- BEM-style CSS class naming convention
